Module 5 Build - tami moore
Indie Film Collective is a multi-page webpage site to let people know aobut the Indie Film Collective.
The webpage tells about the film screenings people can watch, the workshops people can have hands-on experience with film, and the collaborations with each other to create different films.
There are useful tips for attending the film festival.

Website pages:
    Home - Information about the festival
    Film Screenings - Information about the different films
    Workshops - Information about joining a workshop
    Collaborations - Information about getting together to share ideas
